The Vermont Vernal Pool Atlas

The Vermont Vernal Pool Atlas is an online, interactive database that compiles and archives data for the Vernal Pool Mapping Project and the Vernal Pool Monitoring Project. It also provides public access to these important data sets, and allows users to contribute data to these ongoing initiatives.

VPAtlas is currently under development in Alpha release. The data displayed are not final or ready for public use. VPAtlas will enter Beta development in Fall, 2019, during which it will be opened to a small group of public users. When VPAtlas reaches General release (expected 1/1/2020) all features of VPAtlas will be available.

The VPAtlas Project is funded by a State Wildlife Grant from Vermont Fish & Wildlife, with matching grants from the Vermont Watershed Grants Program and the Lintilhac Foundation.
