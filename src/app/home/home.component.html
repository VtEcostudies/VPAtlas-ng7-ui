The Vermont Vernal Pool Atlas

The Vermont Vernal Pool Atlas is an online, interactive database that compiles and archives data for the Vernal Pool Mapping Project and the Vernal Pool Monitoring Project . It also provides public access to these important data sets, and allows users to contribute data to these ongoing initiatives.

VPAtlas is currently under development in an Alpha product release phase. The data displayed are not final and are not ready for public use. VPAtlas will enter Beta development in the Fall of 2019. During its Beta phase, VPAtlas will be opened to a small group of public users. When VPAtlas reaches General release, currently slated for 1/1/2020, all public features of VPAtlas will become available.

The VPAtlas Project is funded by a State Wildlife Grant from Vermont Fish and Wildlife , with matching grants from the Vermont Watershed Grants Program and the Lintilhac Foundation.
