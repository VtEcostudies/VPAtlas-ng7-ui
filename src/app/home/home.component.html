The Vermont Vernal Pool Atlas

Welcome to the Vermont Vernal Pool Atlas (VPAtlas), an online, interactive database that compiles and archives data for VCE's Vermont Vernal Pool Mapping Project and, eventually, the Vermont Vernal Pool Monitoring Project .

You can browse real-time data about suspected and confirmed vernal pools by using the interactive map on the 'Search Pools' tab in the menu bar, where you can also filter your search (e.g., by town). If you'd like to contribute field data and add pool locations, login (or register if you're a first-time user) and the "Add Pool Data" menu tab will appear. You'll find instructions and tips on entering data in that portal - look for the symbol near most fields for more info. To learn what field data to submit, go to the "Field-Data Guides" menu tab to download a data sheet, data sheet instructions, and an amphibian identification guide. If you have any questions or suggestions, please contact us at vpatlas@vtecostudies.org.

VPAtlas is currently under development in Beta release. The data displayed are reviewed by VPAtlas administrators on a regular basis. Beta development will continue and is expected to be complete by the end of 2020.

The VPAtlas Project is funded by a State Wildlife Grant from Vermont Fish & Wildlife , with matching grants from the Lintilhac Foundation and the Vermont Watershed Grants Program.
